## Psychological Thinking Skills (PTS)

The course develops 6 core skills with progressive complexity:

1. **PTS-1**: Scientific Reasoning & Research Literacy
2. **PTS-2**: Application of Psychological Principles
3. **PTS-3**: Biopsychosocial Integration
4. **PTS-4**: Ethical Analysis & Professional Standards
5. **PTS-5**: Data Literacy & Visual Analysis
6. **PTS-6**: Critical Evaluation of Psychological Claims

Each skill is introduced, practiced, and mastered across multiple modules.

## Course Learning Objectives (CLOs)

**CLO1**: Explain the major concepts, theoretical perspectives and historical trends in psychology

**CLO2**: Identify basic research methods in psychology and distinguish evidence quality

**CLO3**: Demonstrate scientific analysis and critical evaluation of psychological claims

**CLO4**: Apply psychology principles to personal, social, and organizational issues

**CLO5**: Demonstrate understanding of biopsychosocial integration and ethical analysis
